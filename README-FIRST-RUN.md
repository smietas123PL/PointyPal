# PointyPal First Run

1. Run `PointyPal.exe`.
2. Follow the **Guided Setup Wizard** to configure your Worker and test your hardware.
3. Once complete, PointyPal will be ready in the tray.
4. Open the Control Center from the tray icon for advanced settings.
5. Visit the **Getting Started** tab in the Control Center for tutorials.

Normal Mode uses real Worker-backed AI/STT/TTS providers only. Fake providers are hidden unless Developer Mode, Safe Mode, or offline self-test mode is active.

Framework-dependent packages require the .NET 8 Desktop Runtime. If the app does not start, install or repair the Microsoft .NET 8 Desktop Runtime and try again.
