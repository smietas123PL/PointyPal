# PointyPal Hotkey Reference

Master these hotkeys to interact with PointyPal efficiently.

## Primary Interaction Keys

| Hotkey | Action | Description |
| :--- | :--- | :--- |
| **Right Ctrl** | **Voice Ask** | Hold to talk to PointyPal. Release to send the request. |
| **Ctrl + Space** | **Quick Ask** | Open the text prompt to type your question. |
| **Escape** | **Cancel** | Instantly stop any ongoing interaction, voice capture, or pointer movement. |

## Visibility & Interface

| Hotkey | Action | Description |
| :--- | :--- | :--- |
| **F9** | **Diagnostics Overlay** | Toggle the status overlay. (Shows compact status in Normal Mode). |
| **F11** | **Full Diagnostics** | Toggle the full developer diagnostics panel (Developer Mode only). |
| **Alt + F4** | **Close Window** | Close the Control Center or Quick Ask window. |

## Developer Mode (Advanced)

*When Developer Mode and Hotkeys are enabled:*

*   **F8**: Trigger fake interaction (assist).
*   **F10**: Toggle between Real and Fake providers.
*   **F12**: Force an immediate screen capture and UIA dump.

---

> [!TIP]
> You can practice these hotkeys in the **Help** tab of the Control Center using the interactive practice cards.
